# trading-notebooks
The repository contains jupyter notebooks with trading content.

## List of notebooks
* TradingViewImagesDownloader
* PlaceTradesBybit

## Installation
```commandline
make clean
make prepare
```

## Usage
**Run job**
```commandline
make-run
```
